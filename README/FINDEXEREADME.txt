pfindexe.exe:

     主要用來判斷所要開啟的執行檔是否己經開啟了, 如果是則叫到最前面來,但其視窗不能縮小否則抓不到

 例如:
      d:\AP\pfindexe.exe   C:\WINDOWS\system32\calc.exe "小算盤"

第一個參數為應用程式即用這個程式去呼叫別人,一定要給後二個參數,否則不會執行
第二個為要執行的應用程, 如果有參數要下要用 " " 包起來
第三個參數為要執行應用程式的 標題, 例如"小算盤"  這個readme 則為 "README.txt-記事本" 也要用" "包起來

      這個程式可以供其它程式以 command line 的方式呼叫 那則第一項為執行檔,二 三項為參數.


如果設成如下則是關掉已開啟的視窗
 
           d:\AP\pfindexe.exe   CLOSE  DIO48
